# AI-Career-Recommmendation-System

## Overview ##
The AI Career Recommendation System is a machine learning based project that recommends suitable career paths for students based on their skills, interests, and resume text. The system uses regression for salary prediction, clustering for student grouping, and natural language processing for resume analysis. The project is implemented using modular programming where each AI component is implemented in separate modules and controlled by a main controller file.

## Features ##
1)Salary prediction using Linear Regression
2)Student grouping using Clustering
3)Resume analysis using Natural Language Processing
4)Career recommendation system
5)CSV dataset handling
6)Modular Python architecture

## Technologies used ##
1)Python
2)Machine Learning
3)Linear Regression
4)Clustering
5)Natural Language Processing
6)CSV File Hand

## Steps to install and run ##

1)Download / Copy all project files
2)Make sure the following files are in the same folder:

main.py
ml_models.py
clustering.py
nlp_module.py
data/students.csv

3)Open a Terminal / Command Prompt
4)Navigate to the project folder:

cd path/to/project

5)Run the main program:

py main.py

6)Follow the on-screen instructions.
